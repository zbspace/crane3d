## 数据查询请求

向 Ros Agent 查询数据

前端接口：/dataController/getData

#### 请求参数：

| 名称      | 值类型 | 描述             |
| --------- | ------ | ---------------- |
| dataType  | string | 操作类型         |
| craneId   | string | 操作台连接设备号 |
| craneType |        |                  |

#### 反馈消息统一格式：

| 名称 | 值类型 | 描述    |
| ---- | ------ | ------- |
| code | int    | code 码 |
| msg  | string | 描述    |
| data | object | 数据    |

### 001 查询船信息状态

查询岸桥船信息状态

`dataType`= shipInfo

#### 返回数据定义：

| 名称                 | 值类型 | 描述                                                    |
| -------------------- | ------ | ------------------------------------------------------- |
| hasShip              | int    | 是否有作业船，0=没有 1=有                               |
| rowCount             | int    | 总排位数                                                |
| visitId              | string | 船期号                                                  |
| vspCode              | string | 干线支线编码                                            |
| shipDirection        | int    | 停靠方向，0=左舷**（船头向左）** 1=右舷**（船头向右）** |
| shipType             | int    | 船类型，0=驳船，1=大船                                  |
| location             | int    | 停靠挡位，0=里，1=中，2=外                              |
| startPos             | int    | 开始小车位，米                                          |
| endPos               | int    | 结束小车位，米                                          |
| currentBayCabinState | int    | 当前贝舱状态 1=舱下 2=舱上 0=无                         |
